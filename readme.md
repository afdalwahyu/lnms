## lnms - Laravel NMS

Network Management System based on Laravel web application framework.

## System Requirements

Tested on CentOS 6
* MySQL 5.5 (rpm from dev.mysql.com)
* PHP 5.5 (yum from webtatic.com)
* fping (need turn off SELinux)
* net-snmp 5.5
